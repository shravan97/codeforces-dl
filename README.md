# :email: Codeforces-dl
## Description
Codeforces-dl helps you download specific problems/problem sets from [codeforces.com](http://codeforces.com)  
## Installation
### From source :
- ``` git clone https://github.com/shravan97/codeforces-dl.git ```  
- ``` cd codeforces-dl ```  
- ``` python setup.py ```
